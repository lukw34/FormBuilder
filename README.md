# form-builder
### 1. Description
Application to build form with question.

### 2. Installation
```bash
yarn
```
### 3. Running (defined in package.json)
- build react application with webpack
```bash
yarn build 
```
- start webpack server with development mode
```bash
yarn start:dev:server
```

- execute unit tests
```bash
yarn test
```
- execute eslint check
```bash
yarn eslint
```
